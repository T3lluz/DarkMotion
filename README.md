# DarkMotion

![DarkMotion Logo](path_to_logo.png) <!-- Add a path to your logo -->

A dark, animated theme for [Stremio-Enhanced](https://github.com/REVENGE977) by REVENGE977.

## Features

- **Dark Theme**: Experience a sleek, dark interface that's easy on the eyes.
- **Animated Elements**: Enjoy smooth animations that enhance the user experience.
- **Customizable**: Easily configure the theme to match your preferences.

## Installation

1. **Clone the Repository**:
    ```sh
    git clone https://github.com/T3lluz/DarkMotion.git
    ```
2. **Navigate to the Directory**:
    ```sh
    cd DarkMotion
    ```
3. **Install Dependencies**:
    ```sh
    npm install
    ```
4. **Run the Application**:
    ```sh
    npm start
    ```

## Usage

- **Activate the Theme**: Follow the instructions in the [Stremio-Enhanced documentation](https://github.com/REVENGE977) to activate DarkMotion.
- **Customize Settings**: Modify the configuration file located at `config/settings.json` to customize the theme.
